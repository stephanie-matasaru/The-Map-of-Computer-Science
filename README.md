## About the Project
This project serves as a visual guide to the vast ecosystem of Computer Science. It breaks down complex domains—from the foundational hardware to the frontiers of AI—into an explorable, galactic map.

### Creative Process (No AI)
This project is a result of manual digital artistry and composition. No AI generation was used in the creation of these visuals.
* **Hand-Drawn & Composited:** The planets and celestial bodies are a mix of custom digital drawings and modified photography.
* **Custom Background:** The sun and space environments were digitally painted to create a cohesive theme.
* **Tooling:** Built and prototyped entirely within **Figma**.
